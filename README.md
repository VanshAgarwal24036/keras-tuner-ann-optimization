# ANN Hyperparameter Tuning using Keras Tuner

This project demonstrates how to optimize an Artificial Neural Network (ANN) using **Keras Tuner** to improve model performance.

## 🚀 Overview
Instead of manually guessing hyperparameters, this project uses automated tuning to find the best combination for:
- Learning rate  
- Number of layers  
- Number of neurons  
- Batch size  

## 🛠️ Tech Stack
- Python  
- TensorFlow / Keras  
- Keras Tuner  
- Google Colab  

## 📊 Workflow
1. Load and preprocess dataset  
2. Build baseline ANN model  
3. Define hyperparameter search space  
4. Run Keras Tuner (RandomSearch / Hyperband)  
5. Train model with best parameters  
6. Evaluate performance  

## 📁 Project Structure
```
ann_keras_tuner.ipynb   # Complete implementation (model + tuning + results)
requirements.txt        # Dependencies
README.md               # Project documentation
```

## ⚙️ Installation
```bash
pip install -r requirements.txt
```

## ▶️ Run
Open the notebook in Jupyter Notebook or Google Colab and run all cells.

## 📈 Results
- Model performance improved after hyperparameter tuning  
- More stable training compared to default parameters
